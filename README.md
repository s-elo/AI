# pattern-recognition

## Install dependencies

```bash
pip install -r requirements.txt
```

## Run the PCA

```bash
python main.py pca
```

## Run the LDA

```bash
python main.py lda
```

## Run the GMM

```bash
python main.py gmm
```

## Run the SVM

```bash
python main.py svm
```

## Run the CNN

```bash
python main.py cnn
```
